Attack commands

If you want to watch everything while the script runs, open four PowerShell windows manually and run these in each:

Admin (tab) — start / run script:

cd "M:\CCNS project work\project\iot-botnet-simulation"
# then run the headless script from here
.\start_and_capture_headless.ps1 -Rebuild -CaptureCount 200


MQTT (tab) — see device telemetry:

cd "M:\CCNS project work\project\iot-botnet-simulation"
docker-compose exec mosquitto mosquitto_sub -t 'devices/data'
# (Ctrl+C to stop)


C2 logs (tab) — live C2 output:

cd "M:\CCNS project work\project\iot-botnet-simulation"
docker logs -f iot-botnet-simulation-c2-1 --tail=200


Victim logs (tab) — live HTTP GETs:

cd "M:\CCNS project work\project\iot-botnet-simulation"
docker logs -f victim --tail=200


If publishing fails inside the script, use an extra Admin2 tab to publish manually:

docker-compose exec mosquitto mosquitto_pub -t 'c2/commands' -m 'do_attack'
